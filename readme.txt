## Overview

In this project, we perform customer segmentation using K‑Means clustering on the popular Mall Customers dataset. The project demonstrates how to preprocess data, determine the optimal number of clusters using the Elbow method, apply clustering algorithms, and visualize the segments through an interactive Flask web dashboard.

## Features

- **Data Preprocessing:** Clean and prepare customer data.
- **Clustering Analysis:** Segment customers using K‑Means clustering.
- **Interactive Visualizations:** Generate scatter plots, bar charts, histograms, and box plots with Plotly.
- **Web Dashboard:** Display segmentation results using a Flask-based front end.
